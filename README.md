

## Requirements
- MySQL


## Install

    $ composer install

    $ npm install && npm run dev

    $ php artisan migrate

    $ php artisan db:seed

Create sample data (optional)

    $ php artisan db:demo

Add 100 profile pics named `user1.png` to `user100.png` in `/storage/app/public/images/avatars`.

    $ php artisan storage:link

Cache Blade icons

    $ php artisan icons:cache

## Test

    $ php artisan test
